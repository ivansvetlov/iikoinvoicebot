ТЕЛЕФОН -> ПК: ПОЛНАЯ СПРАВКА

Что это:
  Ты в Termux на телефоне.
  Команды w* работают через SSH на Windows-хосте и/или локально в текущей среде.

Быстрый старт:
  1) wstartgo
  2) whelp
  3) wstatus
  4) wvibe doctor

Связь и база:
  whelp
    Показать эту справку.
  whelp sets
    Показать готовые наборы команд.
  wrefresh [Termux]
    Обновить toolkit: cd repo -> git pull -> install.sh -> source ~/.bashrc.
  wstartgo [Termux]
    Сначала wrefresh, потом wgo.
  wplan "<текст задачи>"
    Положить задачу из телефона в ops/mailbox/inbox.
  wmailbox [status|list|digest|show|termux|pull|pullclip|reply|watch|prompt|handoff|codexclip|flow|flowclip|resolve]
    Управление mailbox: prompt для Codex + получение ответа на телефон + clipboard.
    Подробно по подкомандам см. блок "wmailbox подробно" ниже.
  wphone [help|session|init|ls|attach|run|send|paste|capture]
    Локальное управление tmux-сессией в Termux (для сквозного phone-terminal потока).
    Подробно по подкомандам см. блок "wphone подробно" ниже.
  wtutor [quick|mailbox|phone|duplex]
    Пошаговые практические сценарии с ожидаемым результатом.
  wgo
    Войти в Windows PowerShell (папка пользователя).
  wenter
    Войти в Windows PowerShell (папка проекта).
  wfixssh [Termux]
    Исправить права SSH-ключей и локальные сокеты в Termux.
  wsetip <ip_пк> [Termux]
    Обновить IP Windows-хоста в ~/.ssh/config.
  wssh [Termux]
    Обычный SSH-вход на Windows.
  wcmd "<PowerShell-команда>" [Termux]
    Выполнить одну PowerShell-команду на ПК.

Проект и git:
  wproj
    Показать путь проекта.
  wstatus
    git status -sb + текущая ветка.
  wpull
    git pull --ff-only.

Сервисы (backend/worker/bot):
  wstart [all|backend|worker|bot]
    Запустить сервис(ы).
  wstop [all|backend|worker|bot]
    Остановить сервис(ы).
  wrestart [all|backend|worker|bot]
    Перезапустить сервис(ы).
  wps
    Показать статус сервисов.

Логи и диагностика:
  wtail [backend|worker|bot|file.log]
    Смотреть live-лог.
  wlogs [...]
    Алиас для wtail.
  wdevstatus
    Проверка dev-окружения.
  wmetrics [args]
    Отчет по метрикам (по умолчанию окно 60 минут).
  wsmoke
    Быстрая проверка: dev_status + /health + /metrics/summary.
  wdiag
    Один экран: host + git + services + smoke.

Тесты и деплой:
  wtest
    Запустить unittest.
  wdeploy [--dry-run] [--yes]
    pull -> test -> restart all -> smoke.

Агенты:
  wvibe
    Старт Vibe с автопрогревом контекста.
  wvibe ask --no-bootstrap "<задача>"
    Быстрый одноразовый запрос.
  wtask "<задача>"
    Короткая постановка задачи (алиас для: wvibe ask --no-bootstrap "<задача>").
  wvibe reconnect
    Продолжить последнюю сессию.
  wvshell
    Легкая интерактивная оболочка: одна строка -> один ask-запрос.
    Внутри: /turns N, /mcp on|off, /mcpcmd ...
  wvibe mcp "<точная команда>"
    Выполнить точную команду через MCP bridge.
  wreconnect
    Алиас для wvibe reconnect.
  wmcp "<точная команда>"
    Алиас для wvibe mcp.
  waider
    Запустить aider в проекте (если установлен).

Готовые сценарии:
  wrun monitor
  wrun incident
  wrun recover
  wrun release

Mailbox-цикл (телефон -> Codex):
  1) wplan "<что сделать>"
  2) wmailbox codexclip
  3) открыть чат Codex и вставить prompt из буфера
  4) опционально: wmailbox flowclip (копирует всю пачку команд)

Ответ на телефон из mailbox:
  1) wmailbox pull
  2) или wmailbox pullclip (ответ + копирование в буфер Android)
  3) для автоподхвата новых ответов: wmailbox watch

Phone-terminal цикл (без UI-автоклика):
  1) wphone init
  2) wphone run "команда"
  3) wphone capture 80
  4) при необходимости: wphone attach

Учебный старт (если хочешь пройти руками):
  wtutor quick

wmailbox подробно:
  wmailbox status
    Показать счетчики mailbox (сколько задач во входящих/обработанных).
  wmailbox list
    Показать список файлов задач в inbox.
  wmailbox digest
    Собрать общий файл задач для Codex: ops/mailbox/for_codex.md.
  wmailbox show
    Показать содержимое ops/mailbox/for_codex.md.
  wmailbox prompt
    Показать короткий готовый prompt для Codex.
  wmailbox handoff
    Обновить digest + вывести готовый prompt (для отправки в Codex).
  wmailbox codexclip
    То же, что handoff, и сразу копировать prompt в Android clipboard.
  wmailbox flow
    Показать мини-пачку команд для цикла "телефон -> Codex".
  wmailbox flowclip
    Скопировать эту мини-пачку в Android clipboard.
  wmailbox termux / wmailbox pull
    Показать ответ для телефона из ops/mailbox/for_termux.md.
  wmailbox pullclip
    Показать ответ для телефона и копировать его в Android clipboard.
  wmailbox reply "<text>"
    Записать новый ответ в ops/mailbox/for_termux.md (полезно для теста watch/duplex).
  wmailbox watch [interval_seconds]
    Следить за for_termux.md в цикле, при изменении печатать новый ответ,
    копировать его в буфер и (если доступно) показывать уведомление.
  wmailbox resolve <file1.md> [file2.md ...]
    Переместить обработанные inbox-файлы в processed.

wphone подробно:
  wphone session [name]
    Показать/сменить имя tmux-сессии (по умолчанию: main).
  wphone init
    Установить tmux (если нет) и создать сессию.
  wphone ls
    Показать tmux-сессии.
  wphone attach
    Подключиться к текущей tmux-сессии.
  wphone run "<cmd>"
    Отправить команду в tmux и нажать Enter.
  wphone send "<text>"
    Отправить текст без Enter.
  wphone capture [lines]
    Снять последние строки экрана tmux (по умолчанию: 120).
  wphone paste
    Вставить Android clipboard в tmux и нажать Enter.

wtutor подробно:
  wtutor quick
    Короткий end-to-end цикл: задача -> prompt -> watch -> ответ.
  wtutor mailbox
    Практика mailbox-команд (status/list/digest/reply/pullclip).
  wtutor phone
    Практика управления tmux-сессией через wphone.
  wtutor duplex
    Практика в 2 Termux-сессиях: watch в одной, reply в другой.

Важно про clipboard в Termux:
  Для termux-clipboard-set/get нужны одновременно:
  1) пакет: pkg install termux-api
  2) Android-приложение: Termux:API

Если не работает:
  1) command not found -> source ~/.bashrc
  2) timeout SSH -> wsetip <новый_ip_пк>, затем wssh
  3) просит пароль по ключу -> wfixssh и повторная проверка ключа на Windows
