# Коментарии к код-агентам из PyCharm (2026)

## 1. Claude Code в JetBrains IDEs
Для работы внутри IDE требуется платная подписка,
можно работать из приложения JetBrains,
но ограничения по бесплатному плану.
**Стоимость:** ~18€/месяц (подписка Claude Pro)

**Что нужно:**
- Подписка Claude Pro или аккаунт Anthropic Console
- Плагин Claude Code из JetBrains Marketplace
- Установленный CLI: `npm install -g @anthropic-ai/claude-code`

**CLI Path (Windows):**
```
C:\Users\MiBookPro\AppData\Roaming\npm\claude.cmd
```

## 2. Cursor (cursor.com)

Есть установленное приложение десктоп Cursor
в день было сделано 10 запросов на 2.15 долларов на 4.7 млн токенов
посмотрим когда сбросится лимит, открытой информации по лимитам нет

**Стоимость:**

- $20/месяц (Pro)

## 3. Cline (бывший Claude Dev) — VS Code Extension

**Стоимость:** Бесплатно (платите только за API)

**УДАЛОСЬ ИНТЕГРИРОВАТЬ БЕСПЛАТНЫЙ ПЛАН МИСТРАЛЬ ЧЕРЕЗ АПИ КЛЮЧ**
Также доступен бесплатно Codex из подписки OpenAI, но он также ограничен подпиской
Также можно мониторить бесплатные планы от
других провайдеров (Mistral, DeepSeek, Qwen) и переключаться между ними

**GitHub:** https://github.com/cline/cline

## 4. Continue.dev — Универсальное расширение

**Стоимость:** Бесплатно (open source)

**Что это:** Расширение для VS Code и JetBrains

**Что нужно:**
- Расширение "Continue" из Marketplace
- Свой API ключ или локальная модель

**Плюсы:**
- Работает в VS Code И JetBrains
- Поддержка множества моделей (Claude, GPT, локальные Ollama)
- Можно использовать бесплатные локальные модели
- Open Source

**Минусы:**
- Нужно настраивать самостоятельно
- Локальные модели слабее облачных

**Сайт:** https://continue.dev

## 5. GitHub Copilot
РАБОТАЕТ из AI CHAT

**Стоимость:**
- $10/месяц (индивидуальный)
- Бесплатно для студентов
Бесплатно до определенной квоты, которая не определена публично, после чего требуется подписка
- сбрасывается на начало месяца, но точные лимиты не раскрываются.
- Бесплатно работает на ChatGPT
- Не выходит в интернет, работает на основе локальных данных и кэша


**Сайт:** https://github.com/features/copilot




## 7. Qwen Code (этот ассистент)

**Стоимость:** Бесплатно

**Что это:** AI-ассистент для работы с кодом

**Плюсы:**
- Понимает контекст проекта
- Может читать/писать файлы
- Запускает команды
- Бесплатно

**Минусы:**
- Требует интернет-соединения
- Не работает полностью автономно

**Сайт:** https://qwen.code.qq.com
