# Папка `.veai/workflows/`

Здесь лежат описания воркфлоу для Veai/IDE — маленькие "шпаргалки" и
пошаговые инструкции для агента и разработчика.

Примеры файлов:
- `Диагностика_request_id.md` — как быстро собрать отчёт по заявке.
- `Регрессия_смоук_чек.md` — смоук-чек перед изменениями.
- `Откат_через_git_безопасно.md` — как откатывать изменения через git.
- `Передача_контекста_новому_агенту.md` — что открыть новому агенту.

Идея: любой новый воркфлоу/чек-лист добавляется сюда как Markdown.
