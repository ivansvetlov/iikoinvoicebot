## Проблема: Claude Code в JetBrains требует отдельной аутентификации

Плагин Claude Code в JetBrains работает с Claude Code CLI, который требует отдельной настройки от приложения Claude на рабочем столе.

**Вот что нужно сделать:**

### 1️⃣ Установите Claude Code CLI (если ещё не установлен)

```powershell
npm install -g @anthropic-ai/claude-code
```

### 2️⃣ Аутентифицируйтесь в CLI

Откройте PowerShell/Terminal и запустите:

```bash
claude
```

Вас перенаправит на браузер для OAuth входа. Завершите вход.

### 3️⃣ Проверьте, что CLI работает

```bash
claude --version
# или
claude /doctor
```

### 4️⃣ Настройте путь в JetBrains (если нужно)

- Откройте PyCharm/IntelliJ
- **Settings → Tools → Claude Code → Claude Command**
- Если `claude` не найден: установите полный путь
  
На Windows с Node.js:
```
node C:\Users\MiBookPro\AppData\Roaming\npm\@anthropic-ai\claude-code\bin\claude.js
```

Или проще — убедитесь, что npm глобальная папка в PATH:
```powershell
npm config get prefix
# Добавьте эту папку в PATH системы
```

### 5️⃣ Перезагрузите IDE

После этого плагин должен подхватить аутентификацию.

---

**Если всё ещё не работает**, запустите в терминале IDE:

```bash
claude /doctor
```

Это покажет статус аутентификации.
