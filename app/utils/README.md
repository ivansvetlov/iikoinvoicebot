# Папка `app/utils/`

Утилиты уровня приложения, которые не тянут за собой фреймворк.

Примеры:
- `user_messages.py` — форматирование ответов пользователю, короткие коды заявок.

Сюда можно добавлять небольшие вспомогательные функции, которые:
- переиспользуются в разных местах;
- не зависят от конкретного веб-фреймворка или Telegram-бота.
