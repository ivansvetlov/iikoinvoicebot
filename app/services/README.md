# Папка `app/services/`

Сервисы верхнего уровня ("оркестраторы") и бизнес-логика.

Ключевые файлы:
- `pipeline.py` — главный пайплайн обработки накладной:
  - извлечение текста (через `parsers`),
  - вызов LLM,
  - валидация результата (error_code, детектор мусора, лимиты токенов),
  - (опционально) отправка в iiko.

Дополнительно здесь могут лежать другие сервисные классы и вспомогательные слои,
которые не привязаны к конкретному фреймворку (FastAPI/aiogram), а реализуют
"чистую" бизнес-логику.
