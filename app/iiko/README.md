# Папка `app/iiko/`

Интеграция с iiko.

Здесь лежат модули, которые:
- открывают iiko UI через Playwright (браузер);
- логинятся по сохранённым кредам;
- находят форму ввода накладных;
- загружают товарные позиции, которые вернул пайплайн.

Важно: этот код завязан на конкретную верстку интерфейса iiko (селекторы).
При изменении UI его нужно будет обновлять.
