# Папка `app/parsers/`

Здесь хранятся парсеры/экстракторы, которые работают с файлами накладных.

Типичные задачи:
- извлечение текста из PDF (текстовый слой / сканы);
- чтение DOCX/TXT файлов;
- предобработка текста и выделение табличных блоков.

Примеры файлов:
- `file_text_extractor.py` — общий вход для извлечения текста по типу файла.
- `invoice_parser.py` — эвристический парсер позиций (до или вместо LLM).

Идея: в `parsers/` лежат "низкоуровневые" штуки, которые ничего не знают про Telegram
и HTTP, только про файлы и текст.
